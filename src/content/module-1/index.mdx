Welcome to Module 1! This module covers the fundamental concepts and building blocks of SVG graphics. You'll learn about the seven primitive shapes, how SVGs work, and the essential concepts that form the foundation of all SVG creation.

Each section builds upon the previous one, so we recommend following the order listed above. Start with the introduction to understand the core concepts, then move through each primitive shape, and finally practice with the exercises.
