Welcome to Module 3! This module covers SMIL, SVG's native animation system, and how to animate properties that CSS cannot handle. You'll learn about SMIL's capabilities, timing functions, orchestration, and how to use SMIL in React and real-world projects.

Each section builds upon the previous one, so we recommend following the order listed above. Start with the problem with CSS, then move through SMIL basics, timing, orchestration, React integration, and advanced projects.
