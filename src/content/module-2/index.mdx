Welcome to Module 2! This module teaches you how to bring your static SVG shapes to life using CSS animations and transforms. You'll learn about transitions, transforms, sequencing, and timing functions to create engaging and interactive SVG animations.

Each section builds upon the previous one, so we recommend following the order listed above. Start with the introduction to understand the animation methods available, then move through transitions, transforms, sequencing, and timing functions before practicing with real projects.
