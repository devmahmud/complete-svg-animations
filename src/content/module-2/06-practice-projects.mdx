## Project 2: Loading ring with pulse

Two animations on the same SVG: a sweeping progress ring (using `stroke-dashoffset`) layered with a pulsing core.

<LiveSVG code={`<div class="loader-stage">
  <svg viewBox="0 0 100 100" width="120" height="120">
    <defs>
      <linearGradient id="grad" x1="0%" x2="100%">
        <stop offset="0%" stop-color="#7c5cff"/>
        <stop offset="100%" stop-color="#ffd43b"/>
      </linearGradient>
    </defs>
    <circle class="track"  cx="50" cy="50" r="40" fill="none" stroke="rgba(255,255,255,0.08)" stroke-width="6"/>
    <circle class="sweep"  cx="50" cy="50" r="40" fill="none" stroke="url(#grad)" stroke-width="6"
            stroke-linecap="round" stroke-dasharray="251.2" stroke-dashoffset="251.2"/>
    <circle class="pulse"  cx="50" cy="50" r="14" fill="url(#grad)"/>
  </svg>
</div>

<style>
.loader-stage { display: flex; justify-content: center; padding: 16px; }
.sweep {
  transform-origin: 50px 50px; transform: rotate(-90deg);
  animation: dash 2s ease-in-out infinite;
}
.pulse {
  transform-origin: 50px 50px; transform-box: fill-box;
  animation: beat 2s ease-in-out infinite;
}
@keyframes dash {
  0%   { stroke-dashoffset: 251.2; }
  50%  { stroke-dashoffset: 0; }
  100% { stroke-dashoffset: -251.2; }
}
@keyframes beat {
  0%, 100% { transform: rotate(-90deg) scale(1); opacity: 1; }
  50%      { transform: rotate(-90deg) scale(1.25); opacity: 0.7; }
}
</style>`} caption="Two synchronized loops on shared 2s clock = a layered loader." />

## Key takeaways

1. **CSS transitions** are for state changes (hover, focus). **Keyframes** are for ambient loops.
2. **Transform** is GPU-accelerated; prefer it over animating positional attributes.
3. **`transform-origin`** controls the pivot — change it to change the entire feel.
4. **`animation-delay` + `:nth-child`** stagger any sequence without JavaScript.
5. **Custom `cubic-bezier`** is what separates "polished" from "default."
