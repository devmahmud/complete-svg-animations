# Interactive SVG Animations Course

Transform your static SVG animations course into an engaging, interactive learning experience with this React application.

## 🚀 Features

### Interactive Learning Experience
- **Live Code Playgrounds** - Write SVG code and see results instantly
- **Progress Tracking** - Monitor your learning progress across modules
- **Interactive Exercises** - Practice with guided exercises and solutions
- **Real-time Preview** - See your SVG animations come to life as you code

### Modern UI/UX
- **Beautiful Design** - Clean, modern interface with smooth animations
- **Responsive Layout** - Works perfectly on desktop, tablet, and mobile
- **Dark/Light Theme** - Comfortable viewing in any lighting condition
- **Smooth Animations** - Framer Motion powered transitions and micro-interactions

### Comprehensive Course Structure
- **4 Complete Modules** - From foundations to advanced techniques
- **Step-by-step Lessons** - Progressive learning path
- **Hands-on Exercises** - Practical coding challenges
- **Solution Checking** - Verify your work against provided solutions

## 📚 Course Modules

### Module 1: SVG Foundations
- Introduction to SVGs and the path model
- The seven primitive shapes (rect, circle, ellipse, line, polyline, polygon, path)
- Coordinate systems and viewBox
- Styling attributes and grouping elements
- Interactive exercises with real-world examples

### Module 2: Your First Animation
- CSS transitions and transforms
- Hover effects and interactive animations
- Animation sequencing and timing functions
- Transform origins and complex animations
- Building animated UI components

### Module 3: Animating the Unanimatable
- SMIL (SVG's native animation system)
- Animating path data and complex attributes
- Path morphing and shape transitions
- Advanced timing and orchestration
- SMIL integration with React

### Module 4: Path and Path Animations
- Advanced path commands and techniques
- Drawing animations with stroke-dasharray
- Motion along paths with animateMotion
- Path morphing and complex transitions
- Real-world project implementations

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd complete-svg-animations
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000` to see the interactive course in action!

### Available Scripts

- `npm start` - Start development server
- `npm run build` - Build for production
- `npm test` - Run tests
- `npm run eject` - Eject from Create React App (not recommended)

## 🎨 Customization

### Adding New Modules
1. Edit `src/context/CourseContext.tsx`
2. Add new module data to the `initialState.modules` array
3. Include lessons and exercises for each module

### Customizing Styles
1. Modify `src/index.css` for global styles
2. Update `tailwind.config.js` for theme customization
3. Edit component-specific styles in individual files

### Adding New Features
1. Create new components in `src/components/`
2. Add new pages in `src/pages/`
3. Update routing in `src/App.tsx`
